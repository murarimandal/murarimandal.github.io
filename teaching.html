KIIT, Bhubaneshwar, India

Natural Language Processing, Spring '24
This course is designed to provide a solid foundation in NLP, covering both theoretical concepts and practical implementation using state-of-the-art tools and frameworks. Here's an overview of what you'll learn: Introduction to NLP: Understand the basics of NLP, including tokenization, stemming, lemmatization, and part-of-speech tagging. Text Representation: Explore different methods for representing text data, such as Bag-of-Words, TF-IDF, word embeddings (e.g., Word2Vec, GloVe), and contextual embeddings (e.g., BERT). Text Classification: Learn how to build text classification models using techniques like Naive Bayes, Logistic Regression, Support Vector Machines, and deep learning approaches such as Convolutional Neural Networks (CNNs) and Recurrent Neural Networks (RNNs). Named Entity Recognition (NER): Dive into NER, a crucial task in NLP for identifying and classifying named entities in text, and learn about popular NER techniques and tools. Sentiment Analysis: Explore sentiment analysis techniques to extract subjective information from text data, including lexicon-based approaches and machine learning-based methods. Sequence-to-Sequence Models: Understand sequence-to-sequence models and how they are used for tasks such as machine translation, text summarization, and question answering. Chatbot Development: Learn how to develop conversational agents (chatbots) using frameworks like TensorFlow and PyTorch, along with popular NLP libraries such as NLTK and spaCy. Advanced Topics: Explore advanced NLP topics such as language modeling, attention mechanisms, transfer learning, and multi-task learning.
